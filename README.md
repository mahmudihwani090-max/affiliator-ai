# 🚀 Alpha Studio - Next.js Version

Platform AI lengkap untuk afiliator TikTok & Shopee Indonesia dengan authentication dan database.

## ✨ Features

- ✅ **Next.js 15** dengan App Router
- ✅ **TypeScript** untuk type safety  
- ✅ **Tailwind CSS** untuk styling
- ✅ **Prisma ORM** dengan SQLite database
- ✅ **NextAuth.js v5** untuk authentication
- ✅ **Gemini AI** integration dengan load balancer
- ✅ **Protected API routes** dengan middleware
- ✅ **User management** dan usage statistics

## 🚀 Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Setup Environment (.env already configured)
Edit GEMINI_API_KEYS dengan API key Anda

### 3. Start Development
```bash
npm run dev
```

Buka http://localhost:3000

## 📖 User Flow
1. Register di /auth/register
2. Login di /auth/login  
3. Dashboard di /dashboard (protected)
4. Use AI studios

## 🚀 Deploy to Vercel
- Zero configuration needed
- Set Root Directory to `nextjs-app`
- Add environment variables
- Deploy!

**Built with ❤️ using Next.js 15, Prisma, and Gemini AI**
